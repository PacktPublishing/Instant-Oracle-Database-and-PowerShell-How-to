About
=====
This script creates an OracleConnection object using Microsoft's Data Provider for Oracle.
This assembly is deprecated and has a number of performance and functionality limitations.

Generally ODP.NET should be used instead; see Chapter 1 for more information.

See http://tiredblogger.wordpress.com/2009/09/01/querying-oracle-using-powershell/ for more information on using System.Data.OracleClient
